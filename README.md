# CodeToCv

CodeToCv is a web platform that allows users to upload their GitHub profile link, apply various filters if desired,
and generate a professional resume based on their GitHub activity and repositories. It leverages GitHub data to
dynamically create resumes, helping users showcase their coding skills and project contributions effectively.

### Installing

* git clone git@github.com:ddosmukhambetov/code-to-cv.git
* cd to project
* make all

This pet-project is currently in active development.
